# ADR 0007: 启用持续 Testnet 实验成交采样

- 状态：Accepted
- 日期：2026-07-14
- 范围：Binance USDⓈ-M Futures Testnet only

## 背景

原三日活动只记录 PA/OF 观察，并用生产准入缺失固定拒绝所有订单，因此无法形成成交、
手续费、止盈/止损命中和净结果样本。Owner 明确要求 Testnet 持续交易以观察数日盈亏，
并明确否决“历史费用后结果为正才恢复 Testnet 下单”的门槛。

## 决策

保留严格基线的生产 `REJECT` 结论，同时增加独立标识的
`TESTNET_EXPERIMENT_OF_PA_V1`。实验最多并行 3 个不同币种，单笔保证金上限 1 USDT、杠杆
上限 10 倍；由主动成交失衡定方向，并要求 book imbalance 或 microprice 至少一项确认。
止损使用最近 5 根 1 分钟结构极值和 ATR 缓冲，距离限定 0.30%–1.20%；止盈为
0.20%–0.35%。每个仓位必须确认交易所原生止损和止盈后才进入持有循环。

任何持仓时长都不触发策略退出。只允许原生止盈、原生结构止损，或操作员停止服务时的
显式平仓。实验结果必须包含手续费后的净值，并标记
`UNVALIDATED_TESTNET_EXPERIMENT`，不得据此承诺收益或解锁生产环境。

## 后果

Testnet 会产生真实模拟成交和可能的连续亏损；每日净亏损 1.00 USDT、每日 100 个样本及
最多 3 个并行仓位限制仍生效。后续参数调整必须基于追加证据中的成交和净结果，而不是只看
胜率或毛收益。
