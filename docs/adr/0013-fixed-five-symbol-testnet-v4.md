# ADR 0013：固定五币池与费用后短目标 Testnet V4

- 状态：Accepted
- 日期：2026-07-15
- 范围：Binance USDⓈ-M Futures Testnet 实验；不解锁生产交易

## 背景

V3 在实际完成的首批样本中费用后为负。当前观测回放还表明，把 35–60 bps 目标机械缩短
到 20 bps 可以提高已平仓胜率，但因止盈净额和结构止损不对称，整体仍为负。用户要求保留
3–5 个固定交易池，以约 1 USDT 保证金、交易所最大杠杆和短目标继续 Testnet 实验。

同名带单员公开订单的可借鉴部分是 BTC/ETH 的高流动性和约 20 bps 毛价格目标；其逆势市价
倍增加仓、全仓保证金和不止损机制不纳入本系统。

## 决策

1. V4 固定交易池为 BTCUSDT、ETHUSDT、BNBUSDT、SOLUSDT、XRPUSDT。服务构造器拒绝其他
   交易池，避免运行参数漂移。
2. 目标毛距离分别为 20、22、25、32、25 bps，反映当前 Testnet 最大杠杆 125x、100x、
   75x、50x、75x 下约 1 USDT 保证金的费用后经济性。
3. 实际下单仍必须通过至少 0.10 USDT 的 taker 手续费及 2 bps 不利滑点后目标检查。不足时
   拒绝，不增加保证金、不倍增加仓。
4. 点差上限从 8 收紧到 5 bps，成交活跃度从近期中位数 0.5 倍提高到 2 倍，相同方向确认从
   2 轮提高到 3 轮。五个活动仓位仍是容量而不是补满目标。
5. 单笔保证金上限保持 1 USDT；结构止损、费用和风险滑点的预计最大净亏损预算提高到
   1 USDT。没有持仓时间到期退出。
6. 生产继续 `RISK_LOCKED`。V4 至少 30 个完成样本前固定为 `INSUFFICIENT_SAMPLE`，即使
   小样本胜率为 100% 也不声明策略有效。

## 后果

V4 的候选会比 V3 更严格，但 BTC/ETH 扩大了低点差机会来源。短目标提高目标触达概率，不能
保证盈利；费用后 Profit Factor、平均盈亏、最大连续亏损和逐币结果仍是主要评价指标。
任何后续降低确认门槛、恢复 DOGE/ADA 或加入补仓都需要新的证据和 ADR。
